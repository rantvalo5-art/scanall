🚀 BB Expansion + Vol + Price Up
NEW
Combo de 3 condiciones simultáneas:
1. BB width se expande rápido (delta absoluto o porcentual)
2. Volumen actual supera el promedio X veces
3. Vela alcista (close > open)
Activar combo BB Expansion Las 3 condiciones deben cumplirse al mismo tiempo
Si el width sube al menos este valor en 1 vela
Ej: de 0.10 → 0.15 (delta = 0.05) ✅
Si el width creció al menos X% relativo
Ej: de 0.10 → 0.13 (+30%) ✅
Volumen de la vela actual > X × promedio últimas 20 velas
Ejemplo que dispara alerta:
width vela anterior: 0.08 → vela actual: 0.14 (delta +0.06, +75%)
volumen: 3.2x promedio | vela: verde ↑

Ejemplo que NO dispara:
width: 0.01 → 0.02 (delta 0.01, insuficiente aunque +100%)
